DC-Stats

A data logging solution for collecting gameplay statistics from Minecraft servers

What is currently tracked
 - Combat
   + Player Deaths
   + PvP Kills
 - Miscellaneous
   + Ender dragons slain
   + withers killed
   + Wardens killed
   + Ravagers killed
   + Evokers killed
   + Piglin Brutes killed
   + Ghasts killed
   + Elder Guardians killed
   + Witches killed
   + Mobs killed
   + Diamonds Mined
   + Emeralds mined
